# Smart_Libary_Sistema_Agendamento_Intelig


O SmartLibrary é um sistema web de agendamento e gerenciamento inteligente para bibliotecas escolares, desenvolvido com o objetivo de automatizar o controle de empréstimos, devoluções e organização de acervos em instituições públicas de ensino. O projeto surgiu a partir da identificação de dificuldades enfrentadas por escolas estaduais na gestão manual de livros, causando atrasos, perdas de exemplares e falta de controle eficiente sobre o patrimônio literário.

A solução foi construída utilizando tecnologias web como HTML5, CSS3, PHP e MySQL, seguindo uma arquitetura cliente-servidor acessível via navegador. O sistema permite maior rastreabilidade do acervo, organização cronológica das devoluções e armazenamento seguro das informações dos alunos e livros, promovendo mais eficiência administrativa e incentivo à leitura no ambiente escolar.

O desenvolvimento do projeto foi baseado na metodologia Design Thinking, com levantamentos presenciais em escolas parceiras para compreender as necessidades reais dos usuários e criar uma solução acessível, intuitiva e de baixo custo.
